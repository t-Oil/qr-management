# QR-Management

## Project Required
```PHP Version 7.4```

## Installing Project
```composer install```

```cp .env.example .env```

```config connection database in .env```

```php artisan key:generate```

```php artisan migrate```

```php artisan db:seed```

## Running Project in Local

```php artisan serve```

## Permissions
```u: admin p: qwerASDF```

```u: user p:qwerASDF```
